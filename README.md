# rnd-zelda-characters

A Random Message About Characters from The Legend of Zelda. It's my first and nonsense package to practice NPM :DD

## Install

```npm
npm install -g rnd-zelda-characters
```

# Usage

```bash
rnd-zelda-characters
```
